这是一个手机休闲小游戏，玩家需要在游戏中扮演一只青蛙进行跳跃旅行 ，在跳跃的过程中 ，会遇到各种各样的障碍物 ，青蛙旅行的距离越长 ，分数越
高 ，失败后通过观看广告复活 ，还可以在排行榜上观察自己的分数，游戏素材来源于网络，项目以学习为主，不可用于商用。
